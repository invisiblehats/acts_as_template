# ActsAsTemplate


## Usage

```ruby
class User < ApplicationRecord
  acts_as_template
end
```

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'acts_as_template'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install acts_as_template
```
